## ASUS ROG Hub - HOMEWORK #2

A simple multi-page HTML/CSS homework project inspired by the ASUS ROG visual style.

## Pages

- `index.html` — Home page
- `services.html` — Services page
- `team.html` — Team page
- `styles.css` — Shared stylesheet (Entry point using @import)

 ## Modular CSS Components
  
* `globals.css` — Base styles, variables, and resets
* `header.css` — Navigation bar and branding styles
* `footer.css` — Copyright and footer layout
* `grid.css` — Flexbox and Grid layouts for cards and sections

## Technologies Used

- HTML5
- CSS3

## How to Run

1. Download or open the project folder.
2. Open `index.html` in your browser.
3. Use the navigation bar to move between pages.

## Purpose

This website was created as an academic exercise to practice:

- HTML page structure
- Navigation between pages
- External CSS styling
- Reusable classes and selectors
- Basic visual consistency across a small website
